# 會員登入註冊系統
##  程式介紹
此程式使用ASP.NET MVC(C#)搭配SQL資料庫撰寫，有基本的登入、註冊、登出功能。
## 註冊頁面

註冊頁面有基本的欄位資料驗證、帳號重複驗證、兩次密碼不相符驗證。
